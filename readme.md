## VS Code Setup

### EsLint

* Install ESLint Extension
* In your workspace or user settings, add:

```json
"eslint.validate": [
    "javascript",
    "javascriptreact",
    "typescript",
    "typescriptreact"
  ]
```

### Prettier

* Install Prettier - Code formatter Extension
* In your workspace or user settings, add:

```json
  "prettier.eslintIntegration": true,
  "editor.formatOnSave": true,
```
